# VIP Статус - Автоматическое Присвоение

## 🌟 Критерии VIP статуса

VIP статус **присваивается автоматически** клиенту при выполнении хотя бы одного из условий:

### Условие 1: По количеству заказов
- **10 или более заказов** → Автоматически становится VIP

### Условие 2: По сумме покупок (LTV)
- **5000 lei или более** потрачено за всё время → Автоматически становится VIP

## 📊 Как это работает

1. **При создании заказа**: система автоматически проверяет, достиг ли клиент критериев VIP
2. **Автоматическое обновление**: если клиент достиг порога, статус меняется на VIP автоматически
3. **Исключение**: клиенты в "Чёрном списке" не получают VIP статус автоматически

## 💡 Использование VIP статуса

### Фильтрация
- На странице **"Клиенты"** можно отфильтровать только VIP клиентов
- Это позволяет быстро найти самых ценных клиентов для:
  - Персональных предложений
  - Специальных акций
  - Индивидуального обслуживания

### Визуальные индикаторы
- VIP клиенты отмечены золотой звёздочкой ⭐
- В таблице клиентов VIP статус выделен оранжевым цветом
- В карточке клиента показывается прогресс до VIP статуса (для обычных клиентов)

### Статистика
- В верхней панели показывается количество VIP клиентов
- Можно отслеживать рост VIP базы

## 🎯 Рекомендации по работе с VIP клиентами

1. **Персональные предложения**
   - Используйте поле "Предпочтения" для заметок о вкусах клиента
   - Предлагайте букеты по их любимым цветам

2. **Приоритетное обслуживание**
   - VIP клиенты могут получать скидки
   - Приоритет при доставке в загруженные дни

3. **Поздравления**
   - Отслеживайте даты рождения VIP клиентов
   - Отправляйте поздравления и специальные предложения

4. **Лояльность**
   - Регулярно просматривайте историю заказов VIP клиентов
   - Благодарите за постоянство и доверие

## 🔄 Изменение статуса вручную

Менеджер может **вручную изменить статус** в карточке клиента:
- **Regular** (Обычный) → базовый статус
- **VIP** → можно присвоить вручную или автоматически
- **Blacklist** (Чёрный список) → проблемные клиенты

**Важно:** Клиенты в чёрном списке не получают VIP автоматически, даже если достигли критериев.

## 🔍 Поиск клиентов

Строка поиска на странице "Клиенты" ищет по:
- **Имени** клиента
- **Email** адресу
- **Телефону** (игнорируя пробелы и форматирование)

Примеры поиска:
- `069123456` найдёт клиента с телефоном `069 123 456`
- `maria` найдёт всех клиентов с именем Maria
- `@gmail` найдёт всех клиентов с Gmail почтой
