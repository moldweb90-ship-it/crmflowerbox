# Модуль «Сотрудники»

## Установка

1. Откройте **Supabase** → **SQL Editor**
2. Выполните скрипт `migrations/create_employees_module.sql`

Миграция:
- Создаёт таблицы `employees` и `shifts`
- Переносит флористов и курьеров в `employees` (те же id)
- Обновляет связи в `sales` (florist_id, courier_id → employees)

## Возможности

### Список (Команда)
- Фото (аватар по первой букве), имя, роль, телефон
- Статус «Сейчас работает»
- Редактирование: ставка, % с продаж

### График (Табель)
- Календарь на месяц
- Клик по ячейке — добавить/убрать смену
- Видно, кто когда работает

### Зарплата
- Ведомость за выбранный месяц
- Ставка × смены + % от продаж
- Итого по каждому сотруднику

## Тарифы в карточке сотрудника

- **Ставка за смену** (lei) — фиксированная сумма за выход
- **% с продаж** — процент от заказов, где сотрудник указан флористом или курьером

## Интеграция с заказами

В форме «Продажа сайт» и «Продажа салон»:
- Курьер — при доставке
- Флорист — при любом типе (кто делал заказ)

Добавление нового сотрудника: `addEmployee` сохраняет в `employees`, после чего он попадает в выпадающие списки.
